TV/Monitor

A monitor is the primary output for a computer. It displays pictures, text, and videos, allowing the user to interact with the computer.

So, the funny thing is I actually got a new TV in the process of making the project. For a while I had been using an old monitor given to me by my grandparents. I'm not sure how old it is, but I can say that it was old enough for the sound to not be working anymore and my computer sometimes not wanting to connect to it. So I was given a TV to use as a replacement of my old one.
